##Introduction
This is port of Notepad++ "Ruby Blue" color scheme to .tmTheme.

tmTheme is used by **Sublime Text** and **TextMate** editor. Currently supported languages are:
* HTML
* CSS
* XML
* PHP
* JavaScript
